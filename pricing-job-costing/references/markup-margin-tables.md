## 2. "What Should I Charge?" Quick Calculator

Walk through this step by step for any service.

### Step 1: Calculate Your Fully Burdened Labor Rate

```
Tech hourly wage:                          $________
Payroll taxes (7.65%):                    +$________
Workers comp insurance (rate varies):     +$________
Health insurance (monthly / 173 hrs):     +$________
PTO cost (days x daily rate / 173):       +$________
Retirement match (if applicable):         +$________
Training (monthly cost / 173):            +$________
Uniforms/PPE (monthly cost / 173):        +$________
─────────────────────────────────────────────────────
FULLY BURDENED HOURLY RATE:                $________
```

**Quick estimate:** Multiply the base wage by 1.25 to 1.45 depending on benefits level.
- Minimal benefits (small shops): Wage x 1.25
- Standard benefits: Wage x 1.35
- Full benefits (health, retirement, PTO): Wage x 1.45

Example: $28/hr wage x 1.35 = $37.80/hr fully burdened

### Step 2: Calculate Your Overhead Per Billable Hour

```
Monthly overhead costs (rent, trucks, insurance,
  office staff, marketing, software, phones,
  tools, licenses, supplies, accounting):  $________

Owner's salary (monthly):                +$________
─────────────────────────────────────────────────────
TOTAL MONTHLY FIXED COSTS:                $________

Number of techs:                           ________
Billable hours per tech per month
  (typically 100-120):                     ________
Total billable hours per month:            ________
─────────────────────────────────────────────────────
OVERHEAD PER BILLABLE HOUR:                $________
  (Total fixed costs / total billable hours)
```

### Step 3: Calculate Your Break-Even Rate

```
Fully burdened labor rate:                 $________
Overhead per billable hour:               +$________
─────────────────────────────────────────────────────
BREAK-EVEN HOURLY RATE:                    $________
```

You MUST charge above this number on every job or you lose money.

### Step 4: Set Your Selling Rate

```
Break-even hourly rate:                    $________
Desired profit margin:                         ____%
─────────────────────────────────────────────────────
SELLING RATE: Break-even / (1 - margin)    $________

Or by desired profit per hour:
Break-even rate + profit/hour:             $________
```

**Example walkthrough:**
```
Tech wage: $28/hr
Burdened rate: $28 x 1.35 = $37.80
Monthly overhead: $16,000
Owner salary: $7,000
Total fixed: $23,000
Billable hours (2 techs x 110): 220
Overhead/hour: $23,000 / 220 = $104.55

Break-even rate: $37.80 + $104.55 = $142.35/hr
Target 35% margin: $142.35 / (1 - 0.35) = $218.99/hr
Rounded rate: $219/hr
```

### Step 5: Calculate a Flat Rate Price

```
SERVICE: _________________________________

Average time to complete (hours):          ________
Labor cost (time x burdened rate):         $________
Average materials cost:                    $________
Overhead (time x overhead/hour):           $________
─────────────────────────────────────────────────────
TOTAL COST:                                $________
Desired margin:                                ____%
FLAT RATE: Cost / (1 - margin):            $________
ROUNDED PRICE:                             $________
```

## 5. Annual Pricing Review Checklist

Perform this review every January (or your fiscal year start). Block 2-4 hours. This is the most impactful thing you do for profitability all year.

### Part A: Cost Review

```
[ ] Pull actual overhead totals for the past 12 months
[ ] Compare to budget — where did costs increase?
[ ] Calculate new monthly overhead figure
[ ] Calculate new overhead per billable hour
[ ] Review fully burdened labor rates for each tech
    - Any wage increases given or due?
    - Workers comp rate change at renewal?
    - Health insurance premium change?
[ ] Review material costs — any major price changes from suppliers?
[ ] Calculate new break-even hourly rate
[ ] Calculate new break-even job count per month
```

### Part B: Margin Analysis

```
[ ] Pull actual gross margin by service category for past 12 months
    - Which services are above target margin?
    - Which are below?
    - Which have the highest volume?
[ ] Calculate overall blended gross margin
[ ] Calculate net margin (after all overhead and owner salary)
[ ] Compare to industry benchmarks (Section 5 of SKILL.md)
[ ] Identify top 5 most profitable service types
[ ] Identify top 5 least profitable service types
[ ] For least profitable: Can they be repriced, or should scope change?
```

### Part C: Competitive Review

```
[ ] Mystery shop 3-5 competitors (or review recent customer intel)
[ ] Document competitor pricing for your top 10 services
[ ] Determine your position: bottom 25%, middle 50%, or top 25%
[ ] Review online cost guides for your area (HomeAdvisor, Angi)
[ ] Check industry association pricing surveys for current year
[ ] Note any competitors who raised or lowered prices
```

### Part D: Price Adjustments

```
[ ] Calculate minimum increase needed to cover cost changes
[ ] Determine target increase (minimum 3-5% annual)
[ ] Update flat-rate price book with new prices
[ ] Update time-and-materials hourly rate
[ ] Update material markup percentages
[ ] Update maintenance plan pricing (see maintenance-programs skill)
[ ] Update any published pricing (website, marketing materials)
[ ] Program new prices into FSM software / price book
```

### Part E: Communication

```
[ ] Draft price increase notification for existing customers
    (see customer-comms skill for templates)
[ ] Set effective date (30+ days from notification)
[ ] Notify maintenance plan members with renewal terms
[ ] Brief sales team / CSRs on new pricing and talking points
[ ] Update any "starting at" pricing on website
[ ] Send price increase letters / emails
[ ] Update commercial contracts at renewal with new rates
```

### Part F: Tracking Setup

```
[ ] Set margin targets by service category for the new year
[ ] Set revenue and job count targets (monthly and quarterly)
[ ] Set break-even tracking cadence (weekly recommended)
[ ] Schedule mid-year pricing check (June/July)
[ ] Schedule next annual review date
```

### Quick Decision Matrix

After completing the review:

| Finding | Action |
|---------|--------|
| Overhead increased, margins held | Raise prices by overhead increase % |
| Overhead stable, margins declined | Material or labor costs rose — reprice affected services |
| Overhead stable, margins strong | Still raise 3-5% (inflation adjustment) |
| Competitor prices significantly higher | Raise to match top 25% of market |
| Competitor prices similar | Differentiate on value, still raise 3-5% |
| Certain services consistently unprofitable | Reprice aggressively or restructure scope |
| Demand exceeds capacity (booked 3+ weeks) | Raise prices 10-20% — you are underpriced |
| Lost bids frequently on price | Don't lower price. Improve value communication. |

## Quick Reference Formulas

Keep these handy:

```
MARKUP %           = ((Selling Price - Cost) / Cost) x 100
MARGIN %           = ((Selling Price - Cost) / Selling Price) x 100
MARKUP TO MARGIN   = Markup / (1 + Markup)
MARGIN TO MARKUP   = Margin / (1 - Margin)
SELLING PRICE      = Cost / (1 - Desired Margin)
BREAK-EVEN (jobs)  = Fixed Costs / (Avg Revenue - Avg Variable Cost)
BREAK-EVEN ($)     = Fixed Costs / Gross Margin %
OVERHEAD/HOUR      = Monthly Overhead / Monthly Billable Hours
BURDENED LABOR     = Wage x 1.25 to 1.45 (depending on benefits)
TRUE HOURLY COST   = Burdened Labor + Overhead Per Hour
```
