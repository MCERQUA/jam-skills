## Submission Best Practices

1. **Use exact NAP** — identical name, address, phone on every listing
2. **Same business description** — write one 150-word description, use it everywhere
3. **Same categories** — use the same service categories across all directories
4. **Same photos** — upload the same core set (logo, team, work) to every platform
5. **Track submissions** in a spreadsheet: directory, URL, date submitted, date verified, NAP correct (Y/N)
6. **Pace submissions** — 5-10 new citations per month (don't submit 50 in one day)
7. **Check aggregators first** — fixing data at the aggregator level fixes dozens of downstream listings
