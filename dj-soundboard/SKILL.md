# DJ Soundboard

**DJ mode ONLY** — the user must explicitly request DJ mode before you use any of these. Do NOT emit `[SOUND:...]` tags in normal conversation.

## Tag

```
[SOUND:name]
```

## Available sounds

`air_horn` · `scratch_long` · `rewind` · `record_stop` · `crowd_cheer` · `crowd_hype` · `yeah` · `lets_go` · `gunshot` · `bruh` · `sad_trombone`

## Usage

Embed inside your spoken response, never alone:

> "And here comes the drop! [SOUND:air_horn]"
