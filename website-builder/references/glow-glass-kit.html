Glow + Glass Kit

Four layered techniques — gradient borders, cursor-spotlight edges, colored glass, ambient bloom. View source; every block is commented.

01 · Gradient glow border
The mask-composite trick A ::before with padding: 1px is painted with a gradient, then masked so only the 1px ring survives. A blurred ::after clone of the same ring is the glow. No real border, no extra markup.

Swap the gradient via the --edge custom property per component — one container class, infinite border styles.

02 · Cursor spotlight border — move your mouse over these

Violet

The border gradient is a radial-gradient centered at --mx/--my, updated by ~6 lines of JS on mousemove.

Ember

A faint white ring underneath keeps the edge readable when the spotlight isn't lit.

Cyan

The ::after adds a low-alpha inner wash that tracks the cursor — depth without noise.
