# Phase 4 — Planning (outline)

**Agent:** article-planning-agent. **Inputs:** all research + Phase-3 summary.
**Output:** `article-outline-planner/article-outline.md` and root `article-plan.md`.

## Outline contents
- **SEO title** (<60 chars, contains `target_keyword`).
- **Meta description** (150-160 chars, compelling, keyword present).
- **Introduction** (200-300 words): hook, problem, what the reader learns, keyword intro.
- **Main sections** — H2/H3 structure. For each: target word count, key points to cover, keywords
  to include (from Phase 2, prioritized by volume/difficulty/intent).
- **FAQ section** — 8-12 questions drawn from `faq-research`.
- **Conclusion** (150-200 words): summary + CTA + final keyword placement.
- **Total:** allocate per-section word counts that sum to `word_target`.

Save the full plan (outline + research sources used + keyword integration plan + link strategy +
word-count breakdown) to root `article-plan.md`. **Immediately proceed to Phase 5** — never ask
"should I continue?".
